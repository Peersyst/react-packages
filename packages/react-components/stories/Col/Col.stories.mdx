# Col

A flexible div that displays its content as a column.

## Usage

```jsx
<Col style={{ width: "100%", height: "300px" }} gap={20} alignItems="center">
    <div style={{ width: "100px", height: "100%", backgroundColor: "red" }} />
    <div style={{ width: "100px", height: "100%", backgroundColor: "blue" }} />
    <div style={{ width: "100px", height: "100%", backgroundColor: "green" }} />
</Col>
```
