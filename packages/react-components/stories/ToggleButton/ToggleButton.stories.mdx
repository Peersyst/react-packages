# Toast

A ToggleButton can be used as a component that indicates a binary state.

## Example

<Canvas>
    <Row justifyContent="center">
        <ToggleButton>Toggle me</ToggleButton>
    </Row>
</Canvas>

## Uncontrolled usage

````jsx
    <ToggleButton>Toggle me</ToggleButton>
````

## Controlled usage

````jsx
    <ToggleButton selected={selected} onChange={handleChange}>Toggle me</ToggleButton>
````
