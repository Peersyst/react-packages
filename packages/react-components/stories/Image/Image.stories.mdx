# Image

Image is an enhanced `<img>` tag that provides an implicit skeleton.

## Example

<Canvas>
    <Row justifyContent="center" gap={50}>
        <Image
            src="https://www.prasm.io/wp-content/uploads/2021/04/Harmony.png"
            alt="harmony"
            style={{ width: "100px", height: "100px" }}
        />
        <Image src="" alt="harmony-not-loaded" style={{ width: "100px", height: "100px" }} />
    </Row>
</Canvas>

## Usage

Usage is the same as a regular `img` element:

```jsx
<Image
    src="https://www.prasm.io/wp-content/uploads/2021/04/Harmony.png"
    alt="harmony"
    style={{ width: "100px", height: "100px" }}
/>
```
