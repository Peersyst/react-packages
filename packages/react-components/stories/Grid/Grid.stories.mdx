# Grid

A responsible and usable grid.

## Usage

Usage is pretty simple, just define the Grid's columns and its breakpoints.

```jsx
<Grid
    cols={3}
    rowGap={60}
    colGap={10}
    breakpoints={[
        { maxWidth: 1200, cols: 2, rowGap: 20, colGap: 20 },
        { maxWidth: 900, cols: 1, rowGap: 30 },
        { maxWidth: 800, cols: 3, rowGap: 10, colGap: 10, justifyContent: "flex-start" },
    ]}
>
    {[...Array(10)].map((e, i) => (
        <div
            key={i.toString()}
            style={{
                width: "200px",
                height: "200px",
                backgroundColor: `rgb(${(i + 2) * 10}, ${i * 24}, ${(i + 1) * 30})`,
            }}
        />
    ))}
</Grid>
```
