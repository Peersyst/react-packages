# Select

Select is used to collect user information from a list of options.

## Example

<Canvas>
    <Select placeholder="Select a value" name="select" expandable>
        <SelectItem value={1}>Hamburger</SelectItem>
        <SelectItem value={2}>Fries</SelectItem>
        <SelectItem value={3}>Apple</SelectItem>
        <SelectItem value={4}>Coconut</SelectItem>
        <SelectItem value={5}>Croissant</SelectItem>
        <SelectItem value={6}>Popcorn</SelectItem>
    </Select>
</Canvas>

## Usage

Selects have SelectItems as their options.

```jsx
<Select placeholder="Select a value" expandable>
    <SelectItem value={1}>Hamburger</SelectItem>
    <SelectItem value={2}>Fries</SelectItem>
    <SelectItem value={3}>Apple</SelectItem>
    <SelectItem value={4}>Coconut</SelectItem>
    <SelectItem value={5}>Croissant</SelectItem>
    <SelectItem value={6}>Popcorn</SelectItem>
</Select>
```

### Multiple

Multiple selection is also supported

```jsx
<Select placeholder="Select a value" expandable multiple>
    <SelectItem value={1}>Hamburger</SelectItem>
    <SelectItem value={2}>Fries</SelectItem>
    <SelectItem value={3}>Apple</SelectItem>
    <SelectItem value={4}>Coconut</SelectItem>
    <SelectItem value={5}>Croissant</SelectItem>
    <SelectItem value={6}>Popcorn</SelectItem>
</Select>
```
