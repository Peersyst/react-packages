## Usage

In order to use this component with the power of typescript, provide the table definition, containing the fields and the type of its cells.

Then, just provide its columns and rows.

```typescript
intrface TableDef {
  id: number;
  name: string;
  price: string;
  day: string;
  week: string;
  marketCap: string;
}
```

```typescript
cont rows: TableDef[] = [
  {
    id: 1,
    name: 'Bitcoin',
    price: '$41,950.05',
    day: '-0.75%',
    week: '-10.72%',
    marketCap: '$791,972,232,024',
  },
  ...,
  {
    id: 10,
    name: 'Polcadot',
    price: '$25.25',
    day: '-1.13%',
    week: '-6.79%',
    marketCap: '$24,895,827,088',
  },
];
```

```typescript
cont columns: TableCol<TableDef>[] = [
  {

```
