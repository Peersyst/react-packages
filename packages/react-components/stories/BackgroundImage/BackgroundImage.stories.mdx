# BackgroundImage

A simple div with a background image and a filter.

## Usage

Usage is pretty simple, just add your background image, and we are good to go!

```jsx
<BackgroundImage
    src="https://blogs.esa.int/space19plus/files/2019/03/nebula.jpg"
    style={{
        width: "100%",
        height: "400px",
        display: "flex",
        alignItems: "center",
        justifyContent: "center",
    }}
>
    <DisplayText>ðŸš€</DisplayText>
</BackgroundImage>
```

You can also add a filter:

```jsx
<BackgroundImage
    src="https://blogs.esa.int/space19plus/files/2019/03/nebula.jpg"
    style={{
        width: "100%",
        height: "400px",
        display: "flex",
        alignItems: "center",
        justifyContent: "center",
    }}
    filter={"linear-gradient(rgba(159, 90, 253, 0.5), rgba(255, 192, 203, 0.5))"}
>
    <DisplayText>ðŸš€</DisplayText>
</BackgroundImage>
```

To style it, just add a style prop or a className:

```jsx
<BackgroundImage
    src="https://blogs.esa.int/space19plus/files/2019/03/nebula.jpg"
    style={{
        width: "100%",
        height: "400px",
        display: "flex",
        alignItems: "center",
        justifyContent: "center",
    }}
    className="background-image"
    filter={"linear-gradient(rgba(159, 90, 253, 0.5), rgba(255, 192, 203, 0.5))"}
>
    <DisplayText>ðŸš€</DisplayText>
</BackgroundImage>
```

```css
.background-image {
    background-position: right;
    background-size: contain;
    background-repeat: repeat;
}
/* Filter is applied in the after pseudo element */
.background-image:after {
    filter: contrast(2);
}
```
