# Popover

The popover is used to display some context on top of another.

## Usage

Usage can be controlled or uncontrolled.

### Controlled usage

Popover state can be controlled with its visible prop and onShow and onHide handlers.

```jsx
<Popover
    showOn="click"
    visible={visible}
    onShow={() => setVisible(true)}
>
    <Popover.Popper>Hello</Popover.Popper>
    <Popover.Content>
          <Button>I own the popover</Button>
    </Popover.Content>
</Popover>
<Button onClick={() => setVisible(!visible)}>
    {visible ? 'Hide popover' : 'Show popover'}
</Button>
```

### Uncontrolled usage

Forget about the popover's state and let it do all the work.

```jsx
<Popover showOn="click">
    <Popover.Popper>Hello</Popover.Popper>
    <Popover.Content>
        <Button>Show popover</Button>
    </Popover.Content>
</Popover>
```
