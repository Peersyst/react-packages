# ProgressBar

Progress bars are used to display determinate or indeterminate progress.

<Canvas>
    <Col gap={20}>
        <ProgressBar />
        <ProgressBar value={70} />
    </Col>
</Canvas>

## Usage

If no value is provided, progress is indeterminate, otherwise it shows the progress value provided.

**Progress value must be between 0 and 100.**

```jsx
<Col gap={20}>
    <ProgressBar />
    <ProgressBar value={70} />
</Col>
```
