# IrregularGrid

Grid's special brother that creates irregular grids.

## Usage

Usage is similar to Grid's. The difference is that now a pattern has to be defined in order to tell the grid how big every cell should be.

The pattern can contain numbers indicating the cell's flex factor in the X axis or arrays of 2 numbers, the first one for the X axis and the second one for the Y axis.

It is called pattern because you don't need to specify every single cell's size, the pattern will be repeated indefinitely.

````jsx
<IrregularGrid
    pattern={[[2, 2], 1, [2, 2], [1, 5], [1, 3], [1, 3], 2, [2, 4]]}
    cols={3}
    colGap={60}
    rowGap={60}
    rowSize="200px"
    breakpoints={
        [
            {
                maxWidth: 1000,
                pattern: [[1, 2], 1, 1, 2, [1, 2], 1, 1, 1],
                cols: 2,
                colGap: 20,
                rowGap: 20,
            },
            {
                maxWidth: 700,
                pattern: [1]
            }
        ]
    }
>
    {[...Array(10)].map((e, i) =>
        <div
            key={i.toString()}
            style={{
                width: "100%",
                height: "100%",
                backgroundColor: `rgb(${(i + 2) * 10}, ${i * 24}, ${(i + 1) * 30})`,
            }}
        />
    )}
</IrregularGrid>
````
