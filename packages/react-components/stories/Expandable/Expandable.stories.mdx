# Expandable

An element that expands its content.

This component uses the slot pattern to achieve an easier customization. So basically, an Expandable must contain the following elements:

### Expandable.Display

This element will always be visible and will toggle the expandable on click.

#### Usage

````jsx
<Expandable.Display>
    Display
</Expandable.Display>
````

<Canvas>
    <Expandable.Display>
        Display
    </Expandable.Display>
</Canvas>

### Expandable.Body

The expandable content that will be shown when the display is clicked.

This element also uses the slot pattern. It is formed by a mandatory Expandable.Content that scrolls, and an optional Expandable.Header and Expandable.Footer.

#### Usage

````jsx
<Expandable.Body>
    <Expandable.Header>Header</Expandable.Header>
    <Expandable.Content>Content</Expandable.Content>
    <Expandable.Footer>Footer</Expandable.Footer>
</Expandable.Body>
````

## Usage

````jsx
<Expandable>
    <Expandable.Display
    >
        Display
    </Expandable.Display>
    <Expandable.Body>
        <Expandable.Header>Header</Expandable.Header>
        <Expandable.Content>Content</Expandable.Content>
        <Expandable.Footer>Footer</Expandable.Footer>
    </Expandable.Body>
</Expandable>
````
