# InfiniteScroll

An infinite scroll component, it handles calls that are made when end is reached.

## Usage

Wrap the infinite content with this component and provide a callback and an end condition to stop making calls.

```jsx
<InfiniteScroll
    callback={handleInfiniteScroll}
    end={noMoreItemsCondition}
    loading={isLoading}
    loadingElement={<ALoadingElement />}
>
    <InfiniteContent />
</InfiniteScroll>
```
