# Hash

Component used to display a <LinkTo kind="components-utils-hash--page#the-component">Hash</LinkTo>.

-   Use length to set how many characters to display.
-   Set the break prop to true to make the hash break and not to scribble in smaller windows.
-   Set an url.

## Usage

#### The component

This component by default, shortens itself with an ellipsis in order to fit in its space.

```jsx
<Hash hash="0x191447fAB0c5736380774CE5197e9Ccda7e835A2" variant="body1" />
```

<Canvas>

#### The break prop

This prop breaks the hash instead of adding an ellipsis.

```jsx
<Hash hash="0x191447fAB0c5736380774CE5197e9Ccda7e835A2" variant="body1" break />
```

#### Set an url

You can also add an url for sending the user to some website (f.e. a blockchain explorer)

```jsx
<Hash
    hash="bc1q35ymugl42k9xlytfg8p7s75cu24me4pz9cu5av"
    variant="body1"
    url="https://blockstream.info/address/bc1q35ymugl42k9xlytfg8p7s75cu24me4pz9cu5av"
/>
```
