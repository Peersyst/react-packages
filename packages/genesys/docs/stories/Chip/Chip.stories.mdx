# Chip

Compact element that displays a small amount  of information related to its context.

<br />

## Variants

The Chip can have a filled or outlined style.

<Canvas>
    <Row gap={20} justifyContent="center" className="with-icon">
        <Chip variant="filled" label="Filled" />
        <Chip variant="outlined" label="Outlined" />
    </Row>
</Canvas>

````jsx
<Row gap={20} justifyContent="center" className="with-icon">
    <Chip variant="filled" label="Filled" />
    <Chip variant="outlined" label="Outlined" />
</Row>
````

<br />

## Rounded

A rounded appearance can be achieved with the rounded prop.

<Canvas>
    <Row gap={20} justifyContent="center" className="with-icon">
        <Chip rounded label="Rounded" />
    </Row>
</Canvas>

````jsx
<Chip rounded label="Rounded" />
````

## Clickable

A Chip can be clickable with the onClick prop.

The clickable prop enables or disabled the clickable appearance but doesn't influence functionality.

<Canvas>
    <Row gap={20} justifyContent="center" className="with-icon">
        <Chip onClick={() => undefined} label="OnClick" />
        <Chip clickable label="Clickable" />
        <Chip clickable={false} onClick={() => undefined} label="Clickable not onClick" />
        <Chip onClick={() => undefined} label="Disabled" disabled />
    </Row>
</Canvas>

````jsx
<Row gap={20} justifyContent="center" className="with-icon">
    <Chip onClick={() => undefined} label="OnClick" />
    <Chip clickable label="Clickable" />
    <Chip clickable={false} onClick={() => undefined} label="Clickable not onClick" />
    <Chip onClick={() => undefined} label="Disabled" disabled />
</Row>
````

<br />

## Deletable

A delete icon appears when onDelete prop is given.

<Canvas>
    <Row gap={20} justifyContent="center" className="with-icon">
        <Chip variant="filled" onDelete={() => undefined} label="Filled deletable" />
        <Chip variant="outlined" onDelete={() => undefined} label="Outlined deletable" />
    </Row>
</Canvas>

````jsx
<Row gap={20} justifyContent="center" className="with-icon">
    <Chip variant="filled" onDelete={() => undefined} label="Filled deletable" />
    <Chip variant="outlined" onDelete={() => undefined} label="Outlined deletable" />
</Row>
````

The delete icon can be customized via the deleteIcon prop.

<Canvas>
    <Row gap={20} justifyContent="center" className="with-icon">
        <Chip variant="filled" onDelete={() => undefined} label="Filled deletable" deleteIcon={<InvalidIcon />} />
    </Row>
</Canvas>

````jsx
<Chip variant="filled" onDelete={() => undefined} label="Filled deletable" deleteIcon={<InvalidIcon />} />
````
