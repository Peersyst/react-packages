# Select

Select is used to collect user information from a list of options.

## Example

<Canvas>
    <Select label="Label" placeholder="Select a value" name="select" expandable options={[
        {value: 1, label: "Hamburger"},
        {value: 2, label: "Fries"},
        {value: 3, label: "Apple"},
        {value: 4, label: "Coconut"},
        {value: 5, label: "Croissant"},
        {value: 6, label: "Popcorn"},
    ]}/>
</Canvas>

## Usage

Select can be used by default with the `options` prop.

```jsx
<Select placeholder="Select a value" name="select" expandable multiple options={[
    {value: 1, label: "Hamburger"},
    {value: 2, label: "Fries"},
    {value: 3, label: "Apple"},
    {value: 4, label: "Coconut"},
    {value: 5, label: "Croissant"},
    {value: 6, label: "Popcorn"},
]}/>
```

<Canvas>
    <Select placeholder="Select a value" name="select" expandable options={[
        {value: 1, label: "Hamburger"},
        {value: 2, label: "Fries"},
        {value: 3, label: "Apple"},
        {value: 4, label: "Coconut"},
        {value: 5, label: "Croissant"},
        {value: 6, label: "Popcorn"},
    ]}/>
</Canvas>

If a higher level of customization is needed, Select also accepts SelectItems as children. One can customize them at will.

```jsx
<Select placeholder="Select a value" expandable>
    <SelectItem value={1}>Hamburger</SelectItem>
    <SelectItem value={2}>Fries</SelectItem>
    <SelectItem value={3}>Apple</SelectItem>
    <SelectItem value={4}>Coconut</SelectItem>
    <SelectItem value={5}>Croissant</SelectItem>
    <SelectItem value={6}>Popcorn</SelectItem>
</Select>
```

### Multiple

Multiple selection is also supported

```jsx
<Select placeholder="Select a value" name="select" expandable multiple options={[
    {value: 1, label: "Hamburger"},
    {value: 2, label: "Fries"},
    {value: 3, label: "Apple"},
    {value: 4, label: "Coconut"},
    {value: 5, label: "Croissant"},
    {value: 6, label: "Popcorn"},
]}/>
```
