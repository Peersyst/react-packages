# Tabs

Tabs component with slots.

Tabs are formed by a TabGroup and TabPanels. TabGroup contains a group of Tabs.

## Usage

### Uncontrolled usage

```jsx
<Tabs>
    <TabGroup indicatorStyle={tabsIndicator} arrowComponent={Arrow}>
        <Tab index={0}>Rocket</Tab>
        <Tab index={1}>Avocado</Tab>
        <Tab index={2}>Penguin</Tab>
    </TabGroup>
    <TabPanel index={0}>
        <Row justifyContent="center">
            <DisplayText>🚀</DisplayText>
        </Row>
    </TabPanel>
    <TabPanel index={1}>
        <Row justifyContent="center">
            <DisplayText>🥑</DisplayText>
        </Row>
    </TabPanel>
    <TabPanel index={2}>
        <Row justifyContent="center">
            <DisplayText>🐧</DisplayText>
        </Row>
    </TabPanel>
    ...
</Tabs>
```

### Controlled usage

```jsx
<Tabs index={value} onIndexChange={handleChange}>
    <TabGroup indicatorStyle={tabsIndicator} arrowComponent={Arrow}>
        <Tab index={0}>Rocket</Tab>
        <Tab index={1}>Avocado</Tab>
        <Tab index={2}>Penguin</Tab>
    </TabGroup>
    <TabPanel index={0}>
        <Row justifyContent="center">
            <DisplayText>🚀</DisplayText>
        </Row>
    </TabPanel>
    <TabPanel index={1}>
        <Row justifyContent="center">
            <DisplayText>🥑</DisplayText>
        </Row>
    </TabPanel>
    <TabPanel index={2}>
        <Row justifyContent="center">
            <DisplayText>🐧</DisplayText>
        </Row>
    </TabPanel>
    ...
</Tabs>
```
