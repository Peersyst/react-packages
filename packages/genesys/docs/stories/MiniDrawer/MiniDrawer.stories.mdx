# MiniDrawer

A mini drawer allows the user to have a clipped drawer that can be expanded or collapsed.

## Usage


```jsx
<>
    <Button onClick={() => setExpanded(!expanded)}>Toggle Mini Drawer</Button>
    <MiniDrawer expanded={expanded}>
        <Col gap={20} style={{ padding: "20px" }}>
            <h3>Item 1</h3>
            <h3>Item 2</h3>
            <h3>Item 3</h3>
        </Col>
    </MiniDrawer>
</>
```
