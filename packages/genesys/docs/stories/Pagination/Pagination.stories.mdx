# Pagination

The Pagination component enables the user to select a specific page from a range of pages.

<Canvas>
    <Col gap={20} alignItems="center">
        <Pagination count={10} />
    </Col>
</Canvas>

## Usage

### Pagination size

```jsx
<Col gap={20} alignItems="center">
    <Pagination count={10} size="sm" />
    <Pagination count={10} />
    <Pagination count={10} size="lg" />
</Col>
```

<Canvas>
    <Col gap={20} alignItems="center">
        <Pagination count={10} size="sm" />
        <Pagination count={10} />
        <Pagination count={10} size="lg" />
    </Col>
</Canvas>

### Buttons

You can enable, disable or customize start, previous, next and last buttons.

```jsx
<Col gap={20} alignItems="center">
    <Pagination count={11} defaultPage={6} startIcon endIcon />
    <Pagination count={11} defaultPage={6} nextIcon={false} previousIcon={false} />
    <Pagination
        count={11}
        defaultPage={6}
        siblingCount={0}
        previousIcon={<WarningIcon />}
        nextIcon={<WarningIcon />}
    />
</Col>
```

### Ranges

```jsx
<Col gap={20} alignItems="center">
    <Pagination count={11} defaultPage={6} siblingCount={0} />
    <Pagination count={11} defaultPage={6} />
    <Pagination count={11} defaultPage={6} siblingCount={0} boundaryCount={2} />
    <Pagination count={11} defaultPage={6} boundaryCount={2} />
</Col>
```
