# SelectorGroup

SelectorGroup is used to choose between boolean inputs (1 or many)

## Example

<Canvas>
    <SelectorGroup
        label="Select yout fav lib/framework"
        name="select"
        options={[
            { value: "react", label: "React" },
            { value: "vue", label: "Vue" },
            { value: "angular", label: "Angular" },
            { value: "svelte", label: "Svelte" },
        ]}
    />
</Canvas>

## Usage

SelectorGroup can be used by default with the `options` prop.

```jsx
<SelectorGroup
    label="Select yout fav lib/framework"
    name="select"
    options={[
        { value: "react", label: "React" },
        { value: "vue", label: "Vue" },
        { value: "angular", label: "Angular" },
        { value: "svelte", label: "Svelte" },
    ]}
/>
```

<Canvas>
    <SelectorGroup
        label="Select yout fav lib/framework"
        name="select"
        options={[
            { value: "react", label: "React" },
            { value: "vue", label: "Vue" },
            { value: "angular", label: "Angular" },
            { value: "svelte", label: "Svelte" },
        ]}
    />
</Canvas>

If a higher level of customization is needed, SelectorGroup also accepts Selector as children. One can customize them at will.

```jsx
<SelectorGroup label="Select your fav food">
    <Selector value={1} label="Hamburger" controller="switch" />
    <Selector value={2} label="Fries" controller="checkbox" />
    <Selector
        value={3}
        label="Coconut"
        LabelProps={{ placement: "top", style: { color: "yellow" } }}
    />
    <Selector value={4} LabelProps={{ placement: "bottom" }} label="Banana" />
</SelectorGroup>
```

### Customize the direction

Switch between displaying the controllers in a row or a col.
You can also pass from the Row/Col props the following parameters:
"gap" | "justifycontroller" | "alignItems" | "reverse"

```jsx
<SelectorGroup
    label="Select yout fav framework"
    direction="row"
    gap="2rem"
    controller="switch"
    name="select"
    options={[
        { value: "react", label: "React" },
        { value: "vue", label: "Vue" },
        { value: "angular", label: "Angular" },
        { value: "svelte", label: "Svelte" },
    ]}
/>
```
