# Switch

A switch lets the user toggle a state in the app.

## Example

    <Row justifyContent="center" gap={10} alignItems="center">

## Usage

Usage can be controlled or uncontrolled.

### Controlled usage

```jsx
<Switch value={value} onChange={handleChange} />
```

### Uncontrolled usage

```jsx
<Switch />
```
