# Checkbox

A basic customizable checkbox.

## Usage

### Default usage

```jsx
<Checkbox />
```

<Canvas>
    <Row justifyContent="center">
        <Checkbox />
    </Row>
</Canvas>

### Custom usage

```jsx
<Checkbox
    checkedIcon={
        <div style={{
            color: "white",
            width: "1.25rem",
            height: "1.25rem",
            border: "1px white solid",
            display: "flex",
            alignItems: "center",
            justifyContent: "center"
        }}>✓</div>}
    icon={<div style={{width: "1.25rem", height: "1.25rem", border: "1px white solid"}} />}
/>
```
