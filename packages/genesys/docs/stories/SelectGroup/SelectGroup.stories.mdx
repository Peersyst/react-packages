# SelectGroup

SelectGroup is used to choose between boolean inputs (1 or many)

## Example

<Canvas>
    <SelectGroup
        label="Select yout fav lib/framework"
        name="select"
        options={[
            { value: "react", label: "React" },
            { value: "vue", label: "Vue" },
            { value: "angular", label: "Angular" },
            { value: "svelte", label: "Svelte" },
        ]}
    />
</Canvas>

## Usage

SelectGroup can be used by default with the `options` prop.

```jsx
<SelectGroup
    label="Select yout fav lib/framework"
    name="select"
    options={[
        { value: "react", label: "React" },
        { value: "vue", label: "Vue" },
        { value: "angular", label: "Angular" },
        { value: "svelte", label: "Svelte" },
    ]}
/>
```

<Canvas>
    <SelectGroup
        label="Select yout fav lib/framework"
        name="select"
        options={[
            { value: "react", label: "React" },
            { value: "vue", label: "Vue" },
            { value: "angular", label: "Angular" },
            { value: "svelte", label: "Svelte" },
        ]}
    />
</Canvas>

If a higher level of customization is needed, SelectGroup also accepts Selector as children. One can customize them at will.

```jsx
<SelectGroup label="Select your fav food">
    <Selector value={1} label="Hamburger" type="switch" />
    <Selector value={2} label="Fries" type="checkbox" />
    <Selector
        value={3}
        label="Coconut"
        LabelProps={{ placement: "top", style: { color: "yellow" } }}
    />
    <Selector value={4} LabelProps={{ placement: "bottom" }} label="Banana" />
</SelectGroup>
```

### Customize the direction

Switch between displaying the controllers in a row or a col.
You can also pass from the Row/Col props the following par√°meters:
"gap" | "justifyContent" | "alignItems" | "reverse"

```jsx
<SelectGroup
    label="Select yout fav framework"
    direction="row"
    gap="2rem"
    type="switch"
    name="select"
    options={[
        { value: "react", label: "React" },
        { value: "vue", label: "Vue" },
        { value: "angular", label: "Angular" },
        { value: "svelte", label: "Svelte" },
```
