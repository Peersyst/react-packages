# AppBar

An app bar with some useful functionalities such as hide on scroll or elevate on scroll.

<Canvas>
    <Row style={{ height: "200px" }} alignItems="center">

                    <Typography variant="body1">Scroll to elevate</Typography>
                </Row>
            </Toolbar>
        </AppBar>
    </Row>
</Canvas>

## Toolbar

It is recommended to use the AppBar with the Toolbar component. This way, we have a better knowledge and consistency of the AppBar's space.

## Usage

Just wrap a Toolbar with an AppBar and add your content inside.

```jsx
<AppBar elevation={0} onScrollElevation={10}>
    <Toolbar>AppBar</Toolbar>
</AppBar>
```
